Write-Up for the HackTheBox Machine - Browsed

001: Port Scan

The provided victim IP 10.10.8.1 was scanned with nmap. Available port found are port 22 and port 80.

nmap -sC -sV -p- 10.10.8.1

002: Host Mapping

To be able to browse the target an IP:Hostname mapping should be made in /etc/hosts. As commonly used by HackTheBox the target was named <victim_machine>.htb and its home page was viewed in browser.
